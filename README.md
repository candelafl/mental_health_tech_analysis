# Mental Health in Tech Survey - Data Analysis

Este repositorio contiene un análisis exploratorio de datos basado en el dataset
*Mental Health in Tech Survey* (2014), realizado como práctica de análisis de datos
utilizando Python y Google Colab (Jupyter Notebook).

## Objetivo
Analizar distintos factores personales y laborales (edad, género, antecedentes
familiares, beneficios laborales, etc.) y su relación con la búsqueda de tratamiento
psicológico en el sector tecnológico.

## Contenido del repositorio
- `mental_health_tech_analysis.ipynb`: Notebook con el análisis completo y conclusiones.
- `mental_health_tech.csv`: Dataset utilizado en el análisis.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab (Jupyter Notebook)
